nn.Module

与其相关的四个模块nn.Parameter，nn.Module，nn.functional，nn.init

  • nn.Parameter 张量子类，表示可学习参数，如 weight，bias
  • 所有网络层基类，管理网络属性
  • 函数具体实现，如卷积、池化、激活函数等
  • 参数初始化方法

8个重要的有序字典

在nn.Module的init中 调用了self._construct 在其之中，一个module相当于一个运算，必须实现forward函数

self._backend = thnn_backend
self._parameters = OrderedDict()
...
self._modules = OrderedDict()

Conv2d也是nn.Module的子类，也有8个OrderedDict，其中_modules是空的，因为没有子模块，而_parameters里面有着Conv2d的参数 weight 和 bias

代码中self.conv2 = nn.Conv2d 的执行会调用到module.py中nn.Module的setattr函数，在函数中依次用isinstance方法来判断类型，然后存储到相应的OrderedDict中

容器

pytorch里面有三个重要容器 nn.Sequential 按顺序包装module nn.ModuleList 像List包装 nn.ModuleDict 像dict一样包装，下面是用nn.Sequential来构建LeNet

self.features = nn.Sequential(
	nn.Conv2d(3, 6, 5),
    nn.ReLU(),
    nn.MaxPool2d(kernel_size = 2, stride = 2),
    nn.Conv2d(6, 16, 5),
    nn.ReLU(),
    nn.MaxPool2d(kernel_size = 2, stride = 2)
)

self.classifier = nn.Sequential(
	nn.Linear(16*5*5, 120),
    nn.ReLU(),
    nn.Linear(120, 84),
    nn.ReLU(),
    nn.Linear(84, classes),
)

Sequential 在 contain.py里面，Sequential继承自Module，在其init中

for idx, module in enumerate(args):
    self.add_module(str(idx), module)

Sequential的forward函数是

def forward(self, input):
    for module in self._modules.values():
        input = module(input)
    return input

上面这样的Sequential每个子网络是用数字来命名的，如果想要自定义名字 那么就要用上OrderedDict

self.features = nn.Sequential(OrderedDict({
    'conv1' : nn.Conv2d(3, 6, 5),
    'relu1' : nn.ReLU(inplace=True),
    ...
}))

ModuleList的三个方法append 在后面加网络层，extend 拼接两个ModuleList insert 指定在ModuleList中插入

class A(nn.Module):
    def __init__(self):
        super(A, self).__init__()
        self.linears = nn.ModuleList([nn.Linear(10, 10) for i in range(20)])
    def forward(self, x):
        for i, linear in enumerate(self.linears):
            x = linear(x)
        return x
# 下面是使用ModuleDict的例子
class B(nn.Module):
    def __init__(self):
        super(B, self).__init__()
        self.choice = nn.ModuleDict({
            'conv' : nn.Conv2d(10, 10, 3),
            'pool' : nn.MaxPool2d(3)
        })
        self.activations = nn.ModuleDict({
            'relu' : nn.ReLU(),
            'prelu' : nn.PReLU()
        })
    def forward(self, x, choice, act):
        x = self.choices[choice](x)
        x = self.activations[act](x)
        return x

卷积

  • 1d 2d 3d 卷积

    一般情况下，卷积核在几个维度上滑动就是几维卷积

  • 卷积 nn.Conv2d

    nn.Conv2d是对多个二维信号进行二维卷积

    in_channels 输入通道数，out_channels 输出通道数，等价于卷积核的个数

    kernel_size 卷积核尺寸 stride 步长 padding 填充个数 dilation 空洞卷积大小

    空洞卷积 就是带孔的卷积，比如5 * 5的卷积核 带孔的话 可能就是1, 1; 1, 3; 1, 5; 3, 1;3, 3; 3, 5; 5, 1;5, 3;5, 5有值

    groups 分组卷积设置 分组卷积 就是分不同部分独立进行卷积, bias 偏置

import struct
def decode_idx3(path):
		bin_data = open(path, 'rb').read()
    _, nums, rows, cols = struct.unpack_from('>IIII', bin_data, 0)
    fmt = '>' + str(nums * rows * cols) + 'B'
    return np.array(struct.unpack_from(fmt, bin_data, 16)).reshape(-1, rows, cols)

下面是LeNet的代码： 

class LeNet(nn.Module):
    def __init__(self, classes):
        super(LeNet, self).__init__()
        self.features = nn.Sequential(
            nn.Conv2d(1, 6, 5)
            ,nn.ReLU(inplace = True)
            ,nn.MaxPool2d(2, 2)
            ,nn.Conv2d(6, 16, 5)
            ,nn.ReLU(inplace = True)
            ,nn.MaxPool2d(2, 2)
        )
        self.classifier = nn.Sequential(
            # 这里是 16 * 5 * 5的原因是 10 * 10 * 16 downsample后成5 * 5 * 16
            nn.Linear(16 * 5 * 5, 120)
            ,nn.ReLU(inplace = True)
            ,nn.Linear(120, 84)
            ,nn.ReLU(inplace = True)
            ,nn.Linear(84, classes)
        )
    
    def forward(self, x):
        x = self.features(x)
        # 这里将[1, 16, 5, 5]的tensor转换成了[1, 400]的tensor
        x = x.view(x.size()[0], -1)
        x = self.classifier(x)
        return x

要注意的是MNIST官网里面的数据集是28 28的 而LeNet的输入要求是32 32

# 数据集的定义
class MNIST(Dataset):
    def __init__(self, data, label, transform):
        super(Dataset, self).__init__()
        self.data = data
        self.label = label
        self.transform = transform
        assert self.data.shape[0] == self.label.shape[0], 'shape incorrect'
        
    def __getitem__(self, idx):
        data = Image.fromarray(np.uint8(self.data[idx]))
        label = self.label[idx]
        data = self.transform(data)
        return data, label
    
    def __len__(self):
        return self.data.shape[0]

def train(epoch):
    net.train()
    loss_list, batch_list = [], []
    for _ in range(epoch):
        for i, (images, labels) in enumerate(tqdm.tqdm(data_train_loader)):
            optimizer.zero_grad()

def test():
    net.eval()
    total_correct = 0
    avg_loss = 0.0
    for i, (images, labels) in enumerate(tqdm.tqdm(data_test_loader)):
        output = net(images)
        labels = labels.to(dtype = torch.int64)
        loss = criterion(output, labels)
        avg_loss += loss.sum()
        pred = output.detach().max(1)[1]
        total_correct += pred.eq(labels.view_as(pred)).sum()

    avg_loss /= 10000
    print('Test Avg. Loss: %f, Accuracy: %f' % (avg_loss.detach().cpu().item(), float(total_correct) / 10000))
